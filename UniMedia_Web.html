🔒 Privacy Policy & Security

Your data protection is our priority

Data Privacy

✅ What We Do

  • Process your data only for the requested AI tools and features
  • Use HTTPS encryption for all data transmission
  • Store user API keys securely using browser-local storage
  • Never sell or share your personal information with third parties
  • Comply with GDPR and international data protection regulations

Security Measures

🔐 How We Protect You

  • Encryption: All API communications use TLS 1.3
  • Authentication: Secure password hashing and session management
  • Data Retention: Processing data deleted after 30 days
  • Access Control: Role-based access and API key validation
  • Monitoring: Regular security audits and vulnerability testing
  • Backup: Regular encrypted backups of user data

Data Usage

📊 What Data We Collect

UniMedia collects and processes:

  • Account information (email, name, password hash)
  • Processing history (metadata about files processed)
  • API usage statistics (for service improvement)
  • Browser information (for technical support)

Your Rights

🛡️ Data Subject Rights

You have the right to:

  • Access your personal data anytime
  • Request corrections to your information
  • Export your data in standard format
  • Delete your account and associated data
  • Restrict processing of your information
  • File a complaint with data protection authorities

Third-Party Services

🤝 External APIs

UniMedia uses trusted third-party services:

  • Groq API: For AI processing (llama model) - Privacy Policy
  • Google Translate: For translation and text-to-speech - Privacy Policy
  • All external services maintain their own privacy standards
